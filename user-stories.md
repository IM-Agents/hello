# User Stories

## Primary Stories
1. As a student, I want to run scientific calculations with trigonometric and logarithmic functions so I can solve academic problems quickly.
2. As a general user, I want a clean calculator interface so I can complete basic arithmetic without confusion.
3. As a developer, I want keyboard shortcuts so I can interact with the calculator efficiently.
4. As a user, I want to switch between degree and radian mode so trig results match my intended context.
5. As a user, I want memory functions so I can temporarily store and reuse values.
6. As a user, I want to view my recent calculations so I can check previous work.
7. As a mobile user, I want the layout to adapt well to my screen so the app remains easy to use on smaller devices.
8. As a user, I want clear error messages so I understand invalid expressions or unsupported math cases.

## Secondary Stories
1. As a user, I want constants like π and e available through buttons so I do not need to type them manually.
2. As a user, I want repeated interactions like backspace and clear to feel consistent and predictable.
3. As a user, I want operator and number buttons visually differentiated so the interface is easier to scan.
