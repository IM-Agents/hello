# Technical Constraints

## Required Technologies
- Frontend must use **React**
- Backend must use **Node.js**

## Dependency Constraints
- Use only minimal, necessary external libraries
- Avoid unnecessary UI or math packages unless clearly justified

## Architecture Constraints
- Frontend should use modular components
- Backend should use clear service/controller separation
- Expression evaluation must be safe and controlled

## UX Constraints
- The design must be responsive for **mobile, tablet, and desktop**
- The UI must support keyboard interaction
- The UI must preserve readability and contrast

## Performance Constraints
- Calculations should feel instantaneous in normal scenarios
- API responses should remain low latency

## Maintainability Constraints
- Code structure should be readable and scalable
- Styling should remain organized and easy to update
- Error handling must be explicit and predictable
